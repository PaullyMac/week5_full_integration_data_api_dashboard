# Week 5 — Full Integration: Data App + API + Dashboard

Monorepo containing:
- **backend-laravel/** — REST API, DB persistence, scheduler poller
- **ai-flask/** — Traccar cache + ETA stub
- **frontend-next/** — Live map UI with proxy

## Quick start (local)

```powershell
cd infra
docker compose up -d --build
```
